# Refresh 기능

사원과 관련된 데이터를 더미 값으로 사용하였지만 

실제로 사원 관련 데이터라면 도중에 이직 등과 같은 이슈로 퇴사를 하게 된다면 데이터 중간의 값이 없어질 수도 있다는 생각을 하여 

Refresh를 누르면 테이블 값을 모두 지우고 API에서 가져온 데이터를 바탕으로 다시 테이블을 작성해주는 방식으로 코드를 작성하였습니다
